## Questions

1. When you first opened the homepage, could you understand what this product does?
2. What do you think is different between Maimang Readbox and a normal feed product?
3. Was registration or login smooth?
4. Do you understand what Authors, Inbox, Later, Collections, and Reading Traces mean?
5. Could you subscribe to an author successfully?
6. Could you read an article successfully?
7. Could you save external reading successfully?
8. Could you create a note or reflection successfully?
9. Do you understand the difference between a note and a reflection?
10. Do you understand what a collection is for?
11. Were there any buttons, labels, or page copy that confused you?
12. Which page felt most like a complete product?
13. Which page felt most unfinished?
14. Does the UI feel quiet, readable, and visually polished enough?
15. Would you keep using this product? Why or why not?
16. Did you see any error, blank screen, loading failure, or wrong redirect?
17. What device and browser did you use?
18. Did you need a proxy or special network setting to open the app?
19. What is the next improvement you most want?
20. Overall score: 1-10.

## Feedback Categories

Use one or more:

- Understanding issue.
- Path issue.
- UI visual issue.
- Copy issue.
- Missing feature.
- Bug.
- Performance issue.
- Access issue.
- Value judgment.

## Notes

- If a bug happened, describe the page, action, expected result, actual result, and whether it repeats.
- Screenshots are useful, but remove private information first.
- Do not send passwords, cookies, tokens, API keys, or private account data.
